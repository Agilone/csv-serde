csv-serde

Hive CSV Support

csv-serde adds real CSV support to hive using opencsv.

Using it

add jar path/to/csv-serde.jar;

create table my_table(a string, b string, ...)
  row format serde 'com.bizo.hive.serde.csv.CSVSerde'
  stored as textfile
;
